Este repositório contém a jornada de aprendizado em **Algoritmos e Estruturas de Dados I**. O foco é o domínio da linguagem C, manipulação de memória e eficiência algorítmica — pilares fundamentais para um Engenheiro de Computação.

---

## 📌 Sobre o Projeto (About)
Este espaço documenta a evolução acadêmica na disciplina de AED I, cobrindo desde a lógica básica até estruturas de dados dinâmicas.

* **Objetivo:** Implementar soluções eficientes, documentar exercícios e entender a linguagem.
* **Destaque:** O repositório inclui o **Projeto Final** da disciplina, que aplica todos os conceitos aprendidos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** C
* **Compilador:** GCC
* **Ambiente:** Windows / CodeBlocks / VSCode

## 📂 Estrutura do Repositório
* `01-Basicos/`: Sintaxe, loops, condicionais e funções.
* `02-Estruturas/`: Listas ligadas, pilhas, filas e ordenação (Sorting).
* `03-Projeto-Final/`: O projeto final da disciplina.
